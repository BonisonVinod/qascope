# Google Pomelli: Social Media & Outreach Prompt Library

Use these pre-tuned prompt blocks directly inside **Google Pomelli (Pomelo)** to generate on-brand, high-converting social creatives, carousels, email visual headers, and WhatsApp cards for the **"Audit the Unreviewed 95%"** launch campaign.

---

## 1. LinkedIn Post + Visual Asset Prompt (Core Campaign Launch)

Copy this block when generating a **LinkedIn Image Ad / Visual Banner** in Pomelli:

```text
Create a premium, minimalist B2B LinkedIn visual banner for QAScope (an AI QA Copilot for support campaigns). 
Theme: "Stop Sampling. Audit the Unreviewed 95%."
Visual Design Rules:
- STRICT DARK MODE. Primary canvas background color must be charcoal-black (#09090B).
- Color accents: Vibrant teal (#14B8A6) and dark teal (#134E4A).
- Center visual: A clean, sleek, glowing glassmorphic card with a teal border overlay. Inside the card, display a simulated database grid showing 100 tiny dot cards. 5 dots are bright glowing teal (label: "5% Reviewed"), and 95 dots are dark, grayed-out zinc (label: "95% Unreviewed & Vulnerable").
- Typography: Bold, high-contrast white sans-serif headline.
- Accent labels: Use red warning borders around the unreviewed 95% section to represent hidden compliance and coaching risks.
- No cartoon characters, no friendly faces, no robots. Visual must feel technical, analytical, and highly operational.
```

Text Overlay: "Stop Sampling. Start Scoring." (Muted gray sans-serif).
Overall aesthetic: Extremely clean, modern SaaS visual, high credibility, developer-grade execution.
